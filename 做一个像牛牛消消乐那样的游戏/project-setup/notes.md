Initialized a strict TypeScript HTML5 Canvas project using Vite. The template enforces zero 'any' types, explicit return types, and single-responsibility modules (Game loop, Input normalization, CanvasRenderer, Configuration). All DOM APIs are guarded with null checks, and the rendering pipeline accounts for devicePixelRatio to ensure crisp visuals on high-DPI screens. ESLint flat config and Prettier are wired together to prevent formatting and type-safety regressions.
