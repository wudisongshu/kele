# kele Core Rules

Read AGENTS.md for project architecture.

## Role
You are building kele, a personal AI assistant project. All code is AI-generated.

## Stack
- TypeScript / Python (depending on module)
- Strict typing always
- Automated testing required

## Constraints
- No manual code writing by human
- No hardcoded secrets
- All public APIs documented
- Critical paths reviewed before commit

## Workflow
1. Plan mode for architecture changes
2. Subagents for exploration
3. Tests before implementation
4. Self-reflect and improve rules after tasks
