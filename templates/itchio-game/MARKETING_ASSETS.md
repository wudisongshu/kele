# itch.io 游戏推广清单

## itch.io 页面优化

- [ ] 封面图：1355x764 px，展示游戏核心画面
- [ ] 5 张以上游戏截图，含 GIF 动图
- [ ] 完善标签（至少 10 个相关标签）
- [ ] 设置"Name your own price"或固定价格
- [ ] 填写系统要求和控制说明
- [ ] 添加 devlog 记录开发过程

## 发布策略

- [ ] 参与 itch.io Game Jam 获取曝光
- [ ] 在 itch.io 社区发布 devlog
- [ ] 请求玩家留评（邮件/页面引导）
- [ ] 设置"Press Kit"供媒体下载

## 外部引流

- [ ] Twitter #indiedev #gamedev 标签
- [ ] Reddit r/gamedev, r/playmygame, r/indiegames
- [ ] IndieDB 页面（如适用）
- [ ] 联系 indie game YouTubers / Streamers
- [ ] 在 Discord indie game 服务器分享
