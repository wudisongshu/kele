# 🚀 itch.io 游戏部署指南

## Step 1: 注册 itch.io 账号
1. 访问 https://itch.io
2. 免费注册开发者账号

## Step 2: 准备游戏包
```bash
# 将游戏打包为 zip
zip -r my-game.zip index.html src/ assets/
```

## Step 3: 创建项目页面
1. 登录 itch.io 进入 Dashboard
2. 点击 "Create new project"
3. 填写游戏信息（标题、描述、截图、封面）
4. 上传 zip 文件
5. 设置定价（免费 / 自愿付费 / 固定价格）

## Step 4: 发布
- 设置为 "Public" 或 "Restricted"
- 支持 HTML5（浏览器直接玩）或可下载版本

## 💰 变现
- **自愿付费**（Pay What You Want）— 最推荐，门槛低
- **固定价格** — 一次性购买
- **itch.io 抽成**: 默认 10%（可自定义为 0%）

## 🎨 推荐配置
- 游戏封面: 630×500px
- 截图: 至少 3 张
- 标签: 准确描述游戏类型
- 描述: 包含玩法说明和特色
