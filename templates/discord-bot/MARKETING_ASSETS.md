# Discord Bot 推广清单

## Discord 生态内

- [ ] 在 top.gg 注册并提交 Bot
- [ ] 完善 Bot 描述、命令列表、邀请链接
- [ ] 在 Discord Bot List (discord.bots.gg) 提交
- [ ] 加入 Discord Bot 开发社区并分享
- [ ] 在目标用户的服务器中（经允许）介绍 Bot

## 外部引流

- [ ] Reddit r/discordbots 发帖
- [ ] Twitter/X 发布 Bot 功能展示
- [ ] Product Hunt（如果是工具型 Bot）
- [ ] 制作"Add to Discord"按钮嵌入网站
- [ ] 写教程博客："如何用 5 分钟部署这个 Bot"

## 留存优化

- [ ] 设置 Bot 状态显示帮助命令
- [ ] 添加 onboarding 消息（首次加入服务器时）
- [ ] 定期发送更新日志到支持服务器
- [ ] 建立 Discord 支持服务器
